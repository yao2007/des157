Project Brief

  • topic summary

    • Dyslexia is a specific learning disability in reading. It also known as reading disorder. It is easy to find many institutions that help patients improve their reading skills, and many website that offer dyslexia treatment. But there are expensive. Therefore, I plan to design an easy-to-use online reading aid modify the text-font that help people with dyslexia and it is for free.

  • target audience description

    • Right now, it is up to 20% may have some degree of symptoms. It has several different diagnosis, which are Central dyslexias, Peripheral dyslexias, and Neglect dyslexia. My target audience is patients who easy to read the letters upsite down.

  • the goals of the project

    • The index page is using for get input links and input text context, then clicking the convert bottom will create a new page with information grabing from the input of the index page. Need to figure out how to get the context form input links.

  • communication strategy

    • When loading page, pop up a tip to tell user what is the purpose of this website. Have less bottom as I can, so that simplify the user steps. About page and tip at loading.

  • goals for the overall look and feel

    • I think the page must be clean and simple so that it does not attract too much attention of patients with dyslexia and stop reading. In my mind, the interface of Google translates is a good example.
